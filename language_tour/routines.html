Routines

BScript has routines as first class citizens, and creates them in the lexical scope. In order to declare one, use the keyword "function" followed by its name, parentheses containing a list of parameters, and a block body:

function printStuff() {
  print "stu" + "ff";
}

This does mean some weird stuff can be done, like the snippet below. I don't know why you would do it, but it's possible, and really weird:

function makeCounter() {
  var a = 0;

  function count () {
    a = a + 1;
  return a;
  }

  return count;
}
var counter = makeCounter();
print counter();
print counter();

var counter2 = makeCounter();
print counter2();

with output:

1
2
1
