Control flow

BScript supports the classics: C style if/else, while, and for statements.

if/else

If the condition inside the "if" parentheses is "truthy", the block following it is executed. If not, and there is an "else" statement following it, the block following it is executed. "Truthy" basically means "anything but null and false". By the way, boolean operations can be done with the "not", "and", and "or" keywords.

if (variable == null) {
  "no value!";
} else if (variable < 0) {
  print "variable is negative number!"
} else if (variable == 2 or variable == 3 or variable == 5 or variable == 7) {
  print "variable is prime smaller than 10"
} else print "variable is something else"

While

Used for repetitions. While the condition between parentheses after "while" is true, it will keep running the program. The first snippet below prints the integers from 0 to 9. The second one creates an endless loop, and the third one never runs.

var i = 0;
while (i < 10) {
print i;
  i = i + 1;
}

while (true) {
  print "help!";
}

var v = 0;
while (v == null) {
  print "Undefined!";
}

For

Syntactic sugar for a very common type of loop. It basically does the same as the first while example, but more compact.

for (var i = 0; i < 10; i = i + 1) print i;

The first parenthesized expression is a variable declaration or assigment, and only runs once, before the first loop. The second expression is equivalent to the while condition, and the third one runs after every loop, but before the second expression is evaluated again.
