Data types and variable declaration

BScript variables are dynamically-typed, and can be declared with the "let" keyword. There are three primitive data types in betascript: functions (numbers are treated as constant functions), strings and booleans values. There is also "null", which represents the lack of a value. In order to print values, use the "print" keyword (yes, it isn't a standard library function. Yes, i know that's weird).

Unassigned variables receive the value null, but it can be directly assigned.
Functions are declared using number literals and functions. The next section will explain more about them.
String literals are declared between quotes.
The only valid boolean values are true and false.
Here are some examples:

var m = null;
var n;
var x = 1.0;
var string = "i am a string";
var otherString = string + " but a little longer";
var y = x + 5;
print otherString;
