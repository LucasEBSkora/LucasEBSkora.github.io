What is Î’Script?

First of all, the language name is read "betascript", because that is a uppercase Beta, which, incidentally, is identical to a uppercase latin alphabet "B".

With that established, BScript is a scripting language based around the concept of mathematical functions as variables. it is an interpreted, semicolon-terminated, lexically-scoped, OO-supporting language.

By mathematical functions, we mean functions defined from a subset of R n to a subset of R. In order to avoid confusion, this documentation will referenced these variables as "functions", and normal programming language functions as routines.
